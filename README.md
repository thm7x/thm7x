# Gitbook
> [GitBook](https://www.gitbook.com/) 是一个基于 Node.js 的命令行工具，可使用 Github/Git 和 Markdown 来制作精美的电子书。

# Environment
macOS Mojave 10.14.4
> 1. node v10.15.3
> 2. npm 6.4.1
> 3. gitBook 3.2.3

# run
> 1. gitbook install 
> 2. gitbook serve
